# 🚗 Prime-Wheels

A **car listing platform** built with React and Vite — a modern, responsive web app where users can browse, list, and explore cars seamlessly.

Live Demo: https://prime-wheels.vercel.app :contentReference[oaicite:1]{index=1}

---

## 🧠 About

Prime-Wheels is a full-stack car marketplace that allows users to:
- Browse car listings
- View detailed car information
- Search and filter by category
- Add listings (planned)
- Handle user accounts and authentication (planned)

This project is designed to demonstrate modern web development practices using React, Vite, Drizzle ORM, Supabase, and other tools.

---

## 📌 Features

✨ Core
- Responsive UI built using React + Tailwind CSS
- Dynamic car listings
- Search & filtering (UI ready)
- Deployment on Vercel

## 📥 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/aniketsharma5102001/Prime-Wheels.git

2. Install dependencies
cd Prime-Wheels
npm install

3. Set up environment variables
Create a .env file and add necessary keys (e.g., Supabase keys, API URLs).
Example:
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_anon_key

4. Run the dev server
npm run dev
